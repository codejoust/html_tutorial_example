The Basics

What is HTML?

  • Stands for HyperText Markup Language.
  • HTML is the building block of the web
  • HTML tells the browser what text to show in the browser
  • HTML provides structure to our websites
  • It also tells the browser where to look for other files.
    • The browser uses the HTML we write to find our images, stylesheets, and script files
    Presenter note for the first slide

How do we write HTML?

  • HTML is written with element tags.
  • Just like in Java, every "function" must be opened and closed (just like braces!)
  • But with HTML, our "braces" are the element tags and the element tag describes the purpose of the text.
  • Let's take a look at what some code would like:
		<h1>This is a header.</h1>
		<p>We use this to create a paragraph.</p>
		<em>For italics.</em> <strong>For bold.</strong>
		
    <ol>
    	<li>
  1. For an ordered list.
    2. 	</li>
    </ol>

		<img src="cover.jpg" alt="cover_photo"/><!--an image tag>

How do we structure HTML?

  • First, we have an <html> tag that encapsulates the entire document. (Think of the brackets of a public class Dog(){)
  • Following that, we have a <head> tag.
    • The head tag should not include text we want on our page
    • We use it as the "instruction guide" for the rest of the page.
  • After that, we have the <body> tag. (That we start only after the head tag has already been close as such </head>.
    • This is where the text of your webpage should go!

What is CSS?

A crash course on CSS

  • CSS stands for "Cascading Style Sheet"
  • It's used for describing how you want your HTML to look - this includes font colors, position of boxes, and layout.
  • CSS is also useful for separate the document content (HTML!) from the presentation of the document itself.
  • Separation is important because sometimes you want to apply the same style to different pages. It also improves readabilit and accessibility.

How do we write CSS?

  • We based it off of selectors in our HTML, using classes:
    •           <div class="logos">
  • You can select the div above like this in CSS:
    •           .logos {
               width: 500px;
               height: 500px;
        }

Example of more CSS

      h1 {
         margin-top: 57px;
         font-size: 60px;
         line-height: 0px;
         font-weight: 400;
         font-family: 'Arvo', serif;
      }

There are multiple ways to select HTML elements.

  1. By class: (<div class="box"> -> .box{})
  2. By name of element(<h1> -> h1 {})
  3. By ID (<li id="clara"> -> #clara{})
  4. These actually have different weighted priorities, and you can use a combination of multiple of them.
  5. Class > element selector

What is Javascript?

An incredible crash course on Javascript.

  • The way we'll be using it is to add interactive design elements to our page.
  • CSS can handle some of this with more selective synatax (#links:hover {})
  • However, JS can handle certain interactivity better, and with better cross-browser compatibility
  • JS looks a lot like Java (same syntax) but they are fundamentally very different.
  • We'll be using JQuery - a JS framework that makes JS much easier to use in a web setting.

Where do we use Javascript?

  • Everywhere!
  • Sliding effects (click to slide down, click to slide up)
  • Page positioning!
  • Slide down to load more tweets
  • Even this slide deck is a JS framework!
  • JS is incredibly powerful - but also pretty complex.
